# Telegram-CDN-Bot
CDN Hosted with Telegram bot as the panel.

## Required
- Nodejs v22 or higher
- Domain

## Featuring
- Session save as .json file
- Upload, Delete, Extract (.zip only), and Move Files
- Telegraf based

## How to Setup
- Download and extract this mf, or you can just clone this repository
- Put your bot token inside from the index.js file
- Do ```npm install``` in your command prompt/terminal
- ```node index.js```
